# To-Do App

A full-stack To-Do application built as part of my Co-Sy development placement.

## Tech Stack

- React
- Vite
- Node.js
- Express
- Firebase Authentication
- Firestore

## Project Goal

Build a secure To-Do application where:

- Users can sign up and log in using Firebase Authentication.
- All to-do data is accessed through a Node.js/Express API.
- Firebase Firestore is only accessed by the backend.
- Each user can only access their own to-do items.
